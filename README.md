# Shrawasti Vijay Dhole — Digital Portfolio

A personal portfolio website showcasing my academic achievements, certifications, projects, and contact information.

## 🌐 Live Site
[View Portfolio](https://shrawastidhole.github.io/portfolio)

## 📁 Folder Structure
```
portfolio/
│
├── index.html          # Main HTML file
├── style.css           # Stylesheet with light/dark mode
├── script.js           # JavaScript (dark mode, lightbox, animations)
├── README.md           # This file
│
└── assets/             # Images and certificates
    ├── cert_python.jpeg        # Python Coursera Certificate
    ├── cert_chemistry.jpeg     # Chemistry Coursera Certificate
    ├── cert_physics.jpeg       # Engineering Physics Certificate
    └── cert_ai.pdf             # Introduction to Modern AI (Cisco)
```

## ✨ Features
- **Dark Mode Toggle** — persistent across sessions (localStorage)
- **Scroll Reveal Animations** — staggered card reveals using IntersectionObserver
- **Image Lightbox** — click any gallery image to view full-screen
- **Responsive Design** — mobile-friendly hamburger menu
- **Smooth Navigation** — active link highlighting on scroll

## 📋 Sections
1. **Home** — Photo, name, introduction, call-to-action
2. **About Me** — Education, interests, goals, strengths
3. **Achievements** — Coursera, Cisco NetAcad, Ai4A, Design Thinking, ACE project certificates
4. **Projects** — C Programming, Discrete Math, Portfolio, Python
5. **Gallery** — All certificates with lightbox viewer
6. **Contact** — Email, GitHub, LinkedIn links

## 🚀 Deployment (GitHub Pages)
1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repository-name`

## 🛠 Tech Stack
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
