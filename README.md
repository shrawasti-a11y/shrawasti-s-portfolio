# Shrawasti Vijay Dhole — Portfolio

A clean, elegant personal portfolio website for Shrawasti Vijay Dhole, B.Tech CSE student at MIT Vishwaprayag University.

## 🚀 Features

- **Dark / Light Mode** toggle with localStorage persistence
- **Custom animated cursor**
- **Smooth scroll animations** using IntersectionObserver
- **Responsive design** — works on mobile, tablet, and desktop
- **Active nav highlighting** as you scroll
- **Image fallback placeholders** if asset images are missing

## 🛠️ Setup

1. **Clone or download** this repository
2. Add your images into the `assets/` folder
3. Update contact details in `index.html` (email, phone, GitHub)
4. Open `index.html` in your browser — done!

## 🌐 Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://yourusername.github.io/repo-name`

## ✏️ Customization

- **Colors**: Edit CSS variables in `:root` inside `style.css`
- **Content**: Edit sections in `index.html`
- **Fonts**: Change Google Fonts import links in `index.html`
